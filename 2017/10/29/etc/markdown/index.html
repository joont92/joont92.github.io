    이스케이프 문자

    태그로 변형되는 문자를 그냥 사용하고 싶을 때가 있습니다.(>, - 등)

    그냥 사용하면 md에서 해석해버리므로 앞에 \ 를 붙여줍니다.

    > : 원래는 인용으로 표시됨

    1998.서울올림픽 : 원래는 1.서울올림픽으로 표시됨

    - : 원래는 리스트로 표시됨
